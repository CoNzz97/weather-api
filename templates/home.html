Weather Data API

This API serves historic weather data.


URL format: http://127.0.0.1:5000/api/v1/station/date

Will return one station at one date: http://127.0.0.1:5000/api/v1/10/1997-02-24

Will return one station and all dates: http://127.0.0.1:5000/api/v1/10

Will return one station and given year: http://127.0.0.1:5000/api/v1/yearly/10/1988
